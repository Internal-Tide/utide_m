reconstruct
===========

.. autofunction:: utide.reconstruct

该函数利用 `solve` 得到的调和常数重构潮汐信号。

参数详解
--------

* **t**: 重构的时间数组。
* **coef**: 由 `utide.solve` 或 `utide.solve_m` 返回的 `Bunch` 数据结构。
* **epoch**: 参考时刻，如果 `t` 是数值天数，则必须提供。
* **verbose**: 是否显示详细输出。
* **constit**: 可选。指定重构中包含的特定调和常数名称列表。
* **min_SNR**: 最小信噪比，默认为 2。
* **min_PE**: 最小能量百分比，默认为 0。

返回值
------

返回一个 `Bunch` 对象，其中包含：

* **h**: 如果是标量时间序列，返回重构后的高度。
* **u, v**: 如果是矢量时间序列，返回重构后的流速分量。
* **t_in**: 输入的时间。
* **t_mpl**: 用于内部处理的转换后时间（days since 0000-12-31）。
